# CLAUDE.md

## Change Ledger

Every time code is modified or new code is added, update [CHANGE_LEDGER.md](CHANGE_LEDGER.md) with:
- **Date** of the change
- **Prompt** that caused the change (the user's request)
- **Model** that made the change (e.g. Claude Opus 4.6, Claude Sonnet 4.6)
- **Summary** of modifications made

Use one date header per day. Within a day, list entries in chronological order (oldest first).
